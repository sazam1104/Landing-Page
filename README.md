#  Thirteen Years of One Direction - Tribute Landing Page

## 📖 Overview

A visually engaging tribute landing page dedicated to celebrating **13 years of One Direction**. The website showcases the band's legacy through a stylish user interface, custom typography, themed visuals, and links to popular music platforms.

This project was built using HTML and CSS to demonstrate frontend development skills, responsive design principles, and creative web design.

---

## ✨ Features

* Modern full-screen landing page
* Custom One Direction-inspired typography
* Background image with immersive design
* Navigation menu with hover effects
* Call-to-action "Let's Go" button
* Music platform social icons
* Clean and attractive user interface
* Custom font integration using `@font-face`

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Custom Fonts
